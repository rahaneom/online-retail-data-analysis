# Online Retail Data Analysis

This project includes performing large-scale data analysis on online retail sales data and analyzing 32 million transactions using `pandas` and `FireDucks` to boost performance.

## 📊 Dataset

The dataset used is the **Online Retail Dataset** from the UCI Machine Learning Repository.

📎 [Click here to access the dataset](https://archive.ics.uci.edu/dataset/352/online+retail)
